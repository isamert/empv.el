#+title: ivjs

Invidious-like API, using YouTube.js. Aims to be a drop in replacement
(to a limited extend) for applications that use Invidious API.

* Usage

Install deno and start ivjs with:

#+begin_src sh
deno run --allow-all main.ts
#+end_src

use it as your Invidious instance:

#+begin_src emacs-lisp
(setq empv-invidious-instance "http://localhost:3534/api/v1")
#+end_src
